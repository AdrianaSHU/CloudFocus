Privacy-First Edge AI

A core principle of this project is privacy. All sensitive video processing is done locally on the Raspberry Pi. The device analyzes facial landmarks and eye aspect-ratio in real-time to determine a user's focus state.

No raw video or identifiable images are ever sent to the cloud. Only the anonymous status (e.g., 'Focused', 'Distracted') and a timestamp are transmitted, ensuring user data is secure and private.

Data-Driven Insights

The anonymised data is sent to a secure cloud backend, built with Django and hosted on Microsoft Azure. This web application provides a powerful dashboard for users to visualize their focus patterns over time.

By seeing when they are most focused or most distracted, users can make informed decisions about their work or study habits, leading to improved productivity and wellbeing.

  • Hardware: Raspberry Pi 4, OpenCV, TensorFlow Lite
  • Cloud: Azure App Service, MySQL, Blob Storage
  • Framework: Django, REST Framework, Plotly.js
